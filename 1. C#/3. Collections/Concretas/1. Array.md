`Array` é uma coleção de elementos do **mesmo tipo**, armazenados em posições sequenciais da memória. Cada elemento é acessado por um índice, que sempre começa em **0**.

Diferente de uma `List<T>`, seu tamanho é definido no momento da criação e não pode ser alterado posteriormente.

---
## 2. Qual problema resolve?

Permite armazenar vários valores relacionados utilizando uma única variável, evitando a criação de diversas variáveis individuais.

É uma estrutura simples e eficiente quando a quantidade de elementos é conhecida previamente e não sofrerá alterações durante a execução da aplicação.

---
## 3. Como funciona?

Um array possui um **tamanho fixo**, definido no momento de sua criação. Cada posição é identificada por um índice numérico, iniciando em **0**.

Os elementos são armazenados de forma sequencial na memória, permitindo acesso rápido por índice.

Por exemplo:

```
string[] alunos =
{
    "João",
    "Maria",
    "Pedro"
};

Console.WriteLine(alunos[1]); // Maria

alunos[1] = "Carlos";

Console.WriteLine(alunos[1]); // Carlos
```

Neste exemplo:

- O array possui três posições.
- Cada elemento é acessado por um índice.
- Os valores podem ser alterados, mas a quantidade de posições permanece a mesma.

---
## 4. Quando usar?

- Quando a quantidade de elementos é conhecida previamente.
- Quando o tamanho da coleção não mudará durante a execução.
- Quando é necessário acessar elementos rapidamente pelo índice.
- Quando a simplicidade da estrutura é suficiente para resolver o problema.

Na prática, arrays são muito utilizados quando o tamanho dos dados é fixo ou quando se busca máxima eficiência no acesso aos elementos.

---
## 5. Exemplo prático

```
string[] meses =
{
    "Janeiro",
    "Fevereiro",
    "Março",
    "Abril"
};

foreach (var mes in meses)
{
    Console.WriteLine(mes);
}
```

Saída:

```
Janeiro
Fevereiro
Março
Abril
```

---
## 6. Principais propriedades e métodos

| Propriedade / Método | Descrição                                                                        |
| -------------------- | -------------------------------------------------------------------------------- |
| `Length`             | Retorna a quantidade de elementos do array.                                      |
| `Array.Sort()`       | Ordena os elementos do array.                                                    |
| `Array.Reverse()`    | Inverte a ordem dos elementos.                                                   |
| `Array.IndexOf()`    | Retorna o índice da primeira ocorrência de um elemento.                          |
| `Array.Exists()`     | Verifica se existe um elemento que atende a uma condição.                        |
| `Array.Find()`       | Retorna o primeiro elemento que atende a uma condição.                           |
| `Array.FindAll()`    | Retorna todos os elementos que atendem a uma condição.                           |
| `Array.Clear()`      | Define os elementos do array para seu valor padrão (`0`, `null`, `false`, etc.). |
| `Array.Copy()`       | Copia elementos de um array para outro.                                          |

---
## 7. Limitações

Apesar de ser uma estrutura simples e eficiente, o array possui algumas limitações.

- O tamanho é fixo após a criação.
- Não é possível adicionar novos elementos com `Add()`.
- Não é possível remover elementos com `Remove()`.
- Inserir ou remover elementos exige criar um novo array ou copiar seus elementos para outro array.
- Possui menos recursos para manipulação quando comparado à `List<T>`.

---
## 8. Resumo

- `Array` é uma coleção de elementos do mesmo tipo.
- Seu tamanho é definido no momento da criação e não pode ser alterado.
- Os elementos são acessados por índice, iniciando em **0**.
- É uma estrutura simples e muito eficiente para acesso aos elementos.
- É indicado quando a quantidade de elementos é conhecida previamente e não sofrerá alterações durante a execução da aplicação.
