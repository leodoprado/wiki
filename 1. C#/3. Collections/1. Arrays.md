1. Oque é?
	É uma coleção de elementos do **mesmo tipo**, armazenados em posições sequenciais da memória. Cada elemento é acessado por um índice, que sempre começa em **0**. Seu tamanho é definido no momento da criação e não pode ser alterado posteriormente.
	
2. Qual problema resolve?
	Permite armazenar vários valores relacionados utilizando uma única variável, evitando a criação de diversas variáveis individuais.
	
3. Como funciona?
	Um array possui um **tamanho fixo**, definido no momento de sua criação. Cada posição é identificada por um índice numérico, iniciando em **0**.

Por exemplo:
```
string[] alunos = { "João", "Maria", "Pedro" };

Console.WriteLine(alunos[1]); // Maria
```

4. Quando usar?
	- A quantidade de elementos é conhecida previamente.
	- O tamanho da coleção não mudará durante a execução.
	- É necessário acessar elementos rapidamente pelo índice.
	- A simplicidade é suficiente para o problema.
	
	Caso seja necessário adicionar ou remover elementos com frequência, normalmente é mais indicado utilizar uma `List<T>`.

5. Exemplo prático

Por exemplo:
```
string[] meses =
{
    "Janeiro",
    "Fevereiro",
    "Março",
    "Abril"
};

foreach (var mes in meses)
{
    Console.WriteLine(mes);
}
```

Limitações:
- O tamanho é fixo após a criação.
- Não é possível adicionar novos elementos com `Add()`.
- Não é possível remover elementos com `Remove()`.
- Inserir ou remover elementos exige criar um novo array ou utilizar métodos da classe `Array`, o que normalmente é menos prático do que utilizar uma `List<T>`.
