1. Oque é?
	Propriedades são uma forma de **controlar o acesso aos dados de uma classe**. Elas funcionam como uma “porta controlada” para ler e alterar valores de atributos.

Exemplo simples:
```
public class Pessoa
{
    public string Nome { get; set; }
}
```

2. Qual problema resolve?
	Se você deixasse os dados públicos diretamente, qualquer parte do código poderia alterar sem controle:
	- valores inválidos
	- regras quebradas
	- estado inconsistente do objeto

Exemplo ruim:
```
public class Conta
{
    public decimal Saldo;
}
```

Qualquer um pode fazer:
```
conta.Saldo = -1000;
```

Propriedades resolvem isso permitindo **controle de leitura e escrita**.

3. Como funciona?
	Propriedades são uma combinação de:
	- get -> leitura
	- set -> escrita

Exemplo:
```
public class Conta
{
    public decimal Saldo { get; private set; }
}
```
Qualquer um pode ler o saldo, só a própria classe pode alterar.

- Propriedade com regra

Você pode adiciona lógica:
```
public class Conta
{
    private decimal saldo;

    public decimal Saldo
    {
        get { return saldo; }
        private set
        {
            if (value < 0)
                throw new Exception("Saldo inválido");
            
            saldo = value;
        }
    }
}
```

4. Quando usar?
	Use propriedades quando:
	- você quer expor dados da classe
	- precisa controlar leitura/escrita
	- quer aplicar regras de validação
	- quer proteger o estado interno do objeto
	
	Exemplos:
	- Nome de usuário
	- Saldo bancário
	- Id de entidade
	- Status de pedido

4. Exemplo prático:

Cenário: Conta bancária
```
public class ContaBancaria
{
    public string Titular { get; set; }
    
    public decimal Saldo { get; private set; }
    
    public void Depositar(decimal valor)
    {
        if (valor <= 0)
            return;
        
        Saldo += valor;
    }
    
    public void Sacar(decimal valor)
    {
        if (valor <= 0 || valor > Saldo)
            return;
        
        Saldo -= valor;
    }
}
```

Utilização:
```
var conta = new ContaBancaria();

conta.Titular = "Leonardo";
conta.Depositar(100);
```

*Resumo mental:*
**Propriedade é uma “variável com controle”**.
Ela parece um campo comum, mas por trás pode ter regras e lógica.
