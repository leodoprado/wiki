1. Oque é?
	É um mecanismo que permite que uma classe reutilize atributos e métodos de outra classe. A classe que fornece as características é chamada de classe pai, e a que herda essas características é chamada de classe filha.
	
2. Qual problema resolve?
	Evita a duplicação de código e facilita o reaproveitamento de funcionalidades comuns entre diferentes classes.

3. Como funciona?
	Uma classe filha herda os membros da classe pai e pode adicionar novos comportamentos ou modificar os existentes.
	
4. Quando usar?
	Quando existe uma relação do tipo **"é um"** entre as entidades.
	
	Exemplos:
	- Um Cliente é uma Pessoa.
	- Um Funcionário é uma Pessoa.
	- Um Cachorro é um Animal.

3. Exemplo prático?
	
Classe Pai:
```
	public class Pessoa
	{
	    public string Nome { get; set; }

	    public void Apresentar()
	    {
	        Console.WriteLine($"Olá, meu nome é {Nome}");
	    }
	}
```

Classe Filha:
```
public class Cliente : Pessoa
{
    public string Email { get; set; }
}
```

Utilização:
```
var cliente = new Cliente();

cliente.Nome = "Leonardo";
cliente.Email = "leonardo@email.com";

cliente.Apresentar();
```
