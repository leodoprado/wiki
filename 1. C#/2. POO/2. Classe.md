É um **molde** ou **modelo** que define quais características (atributos) e comportamentos (métodos) um objeto terá.
### Qual problema resolve?  
Evita a repetição de código e permite criar vários objetos com a mesma estrutura e comportamento
### Como funciona?  
A classe descreve como um objeto deve ser. Quando precisamos utilizar essa estrutura na aplicação, criamos uma instância da classe, chamada de objeto.
### Quando usar?  
Sempre que for necessário representar uma entidade ou conceito do sistema que possua dados e comportamentos próprios.

### Exemplo prático?
```
public class ContaBancaria
{
    public string Titular { get; set; }
    public decimal Saldo { get; set; }

    public void Depositar(decimal valor)
    {
        Saldo += valor;
    }
}
```
