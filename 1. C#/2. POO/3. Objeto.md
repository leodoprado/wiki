1. Oque é?
	É uma **instância de uma classe**. Ele representa uma entidade específica criada a partir do molde definido pela classe.
	
2. Qual problema resolve?
	Permite representar e manipular elementos reais do sistema, cada um com seus próprios dados e comportamentos.
	
3. Como funciona?
	Primeiro definimos uma classe (molde). Depois criamos objetos a partir dela. Cada objeto possui seu próprio estado (valores dos atributos), mas compartilha a estrutura e os comportamentos definidos pela classe.
	
4. Quando usar?
	Sempre que for necessário criar uma representação concreta de uma entidade do sistema.
	
5. Exemplo prático?
```
	var conta = new ContaBancaria();

	conta.Titular = "Leonardo";
	conta.Saldo = 1000;
```
