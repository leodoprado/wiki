1. Oque é?
	Construtor é um **método especial de uma classe** que é executado automaticamente quando um objeto é criado. Ele serve para inicializar o objeto.

Exemplo simples:
```
public class Pessoa
{
    public string Nome { get; set; }

    public Pessoa()
    {
        Nome = "Sem nome";
    }
}
```

2. Qual problema resolve?
	Sem construtores, você criaria objetos “incompletos” ou precisaria configurar tudo manualmente depois.
	
	Isso pode gerar:
	- Objetos inválidos
	- Código repetitivo
	- Esquecimento de inicialização
	- Inconsistência no sistema

Exemplo ruim:
```
var pessoa = new Pessoa();
pessoa.Nome = "Leonardo";
```

Aqui o objeto nasce vazio e depende de você lembrar de configurar tudo.

3. Como funciona?
	Quando você usa `new`, o C#:
	- Aloca memória para o objeto
	- Executa o construtor automaticamente
	- Retorna o objeto pronto para uso

Exemplo:
```
var pessoa = new Pessoa();
```

	Fluxo Interno:
	new Pessoa()
	↓
	Executa construtor Pessoa()
	↓
	Objeto pronto

- Construtor com parâmetros

Você pode obrigar informações na criação do objeto
```
public class Pessoa
{
    public string Nome { get; set; }

    public Pessoa(string nome)
    {
        Nome = nome;
    }
}
```

Utilização:
```
var pessoa = new Pessoa("Leonardo");
```

- Sobrecarga de construtores

Você pode ter mais de um construtor:
```
public class Pessoa
{
    public string Nome { get; set; }
    public int Idade { get; set; }

    public Pessoa()
    {
        Nome = "Desconhecido";
    }

    public Pessoa(string nome, int idade)
    {
        Nome = nome;
        Idade = idade;
    }
}
```

4. Quando usar?
	Use construtores quando:
	- Você quer garantir que o objeto já nasça válido
	- Existem dados obrigatórios para o objeto funcionar
	- Quer evitar configuração manual após criação
	- Precisa padronizar inicialização
	
	Exemplos:
	- Conta bancária (precisa de titular e saldo inicial)
	- Pedido (precisa de cliente)
	- Usuário (precisa de nome/email)

4. Exemplo prático?

Cenário: conta bancária
```
public class ContaBancaria
{
    public string Titular { get; private set; }
    public decimal Saldo { get; private set; }

    public ContaBancaria(string titular, decimal saldoInicial)
    {
        Titular = titular;
        Saldo = saldoInicial;
    }

    public void Depositar(decimal valor)
    {
        Saldo += valor;
    }
}
```

Utilização:
```
var conta = new ContaBancaria("Leonardo", 100);

conta.Depositar(50);
```

Resumo mental:
Construtor é o “momento de nascimento” do objeto. Ele garante que o objeto já comece pronto para uso.
